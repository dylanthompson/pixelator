# pixelator
Pixel art utility

Creates Gif animated gradients and hopefully other animated elements

**Installation**
1) Requires Node and npm
 - https://www.npmjs.com/get-npm

2) Needs Typescript
> npm install -g typescript

3) Install Dependencies
> npm install

**Usage**
1) Transpile
> tsc

2) Run
> node .\pixelator.js
